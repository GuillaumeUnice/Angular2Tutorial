NG Module

Angular Modules vs ES6 Modules

  • Angular modules is a decorator
  • Angular modules can import all Angular concept : Component, Pipe, Service
  • ES6 modules is just the standard to include and use a js file from another : wrap private details in a closure and expose only the public API we want

Rout Module

The Root Module is all we need for Application
Then, we structure in lot of Features Modules (collections of related functionality). And Import the Feature Module into the Root Module

Bubbling Up

When a component ask a dependency there is three way before throw an error:

  • Angular try to find this dependency in property provider of current component
  • Try to find in parent provider, then the parent of parent etc. => Else Angular throw an error
  • The third possibility it search until host component provider and not above
